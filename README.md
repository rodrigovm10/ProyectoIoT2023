## Objetivo general
Este proyecto tiene como objetivo la creación de un dispensador de alimento para una mascota, 
esto con el fin de tener una forma automatizada de tener controlada la alimentación de las mascotas, además de que podrá ser 
de ayuda para tener un más ameno sobre el alimento de la mascota.
